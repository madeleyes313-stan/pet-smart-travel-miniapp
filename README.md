# 宠物智能出行 - 微信小程序

这是“宠物智能出行”项目的小程序端（MVP）。

## 运行
1. 用微信开发者工具打开本目录。
2. 确认 `project.config.json` 的 `appid` 已配置。
3. 在 `app.js` 中确认后端地址 `apiBase`。

## 目录
- `pages/home` 首页（多模态问答入口）
- `pages/trip` 行程规划
- `pages/risk` 风险评估与应急预案
- `pages/pet` 多宠物管理
- `pages/profile` 设置与运营演示模式

## 注意
- `project.private.config.json` 为本地开发工具配置，不提交到仓库。
