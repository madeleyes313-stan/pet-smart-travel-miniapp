# 微信小程序提审说明（v1.0.0）

## 版本信息
- 版本号：`v1.0.0`
- 发布目标：宠物智能出行 MVP 首发版本
- 适用平台：中国大陆微信小程序

## 核心功能
1. 宠物户外知识问答（支持文本问题，预留多模态输入扩展）
2. 智能行程规划（天气、路线、清单、预算建议）
3. 户外风险评估与提醒订阅（风险等级、建议动作、订阅提醒）
4. 宠物档案管理（多宠物信息维护）
5. 应急预案入口（附近宠物医院和急救建议）
6. 运营演示模式（仅开发联调用）

## 体验路径
1. 打开小程序进入“首页”，输入问题并提交问答
2. 切换“行程”页输入出发地和目的地，生成行程
3. 切换“风险”页输入地点，执行风险评估并订阅提醒
4. 切换“宠物”页填写宠物信息并保存
5. 切换“我的”页进行授权设置和可访问性设置

## 隐私与合规说明
- 仅在用户主动授权后使用位置、录音、相册等能力
- 医疗相关输出仅为辅助建议，不替代兽医诊断
- 高风险建议提供就医/应急指引入口

## 备注
- `project.config.json` 中使用 `touristappid` 占位，避免在公开仓库暴露真实 AppID
- 实际发布请在本地微信开发者工具中绑定真实 AppID，不提交到公开仓库
